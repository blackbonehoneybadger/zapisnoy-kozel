# DOFFA — Финальные арт-ассеты маскота-тапалки

Сейчас маскот собран из SVG/CSS/Framer Motion (`src/features/beans/DoffaMascot.tsx`) —
это качественная временная версия. Ниже — что нужно дорисовать художнику для
финального «вау»-уровня. Референс бренда — логотип DOFFA Espresso Bar (чашка с
торчащими ногами, горы, тёплая палитра).

## Палитра (design tokens)

- эспрессо `#16110b` / `#1e1710` / `#2a2016`
- крем `#f3e9ce` / `#f7efd8` / `#f2d9a0`
- золото/медь `#e0a43b` / `#ecba54` / `#bb5c3c`
- лес `#3a5e42` / `#6e7d5b`
- джинсы (ноги) `#3f6f6a` / `#2c5450`
- кофе `#3a2416` / крема `#8a5232`

## Требуемые иллюстрации (слои — раздельные PNG/SVG с альфой)

1. **Чашка** — основной корпус (тело + кромка + ручка), кремово-золотая, лёгкий объём и мягкая тень. Отдельно от кофе и ног, чтобы анимировать независимо.
2. **Кофе (поверхность)** — тёмный кофе + крема-завиток; отдельный слой для «волны/всплеска».
3. **Ноги** (2 шт., джинсы + кроссовки) — свисают через кромку; раздельные слои левой/правой для независимого swing/kick. Кроссовки с белой подошвой и медным акцентом шнурков.
4. **Рука** — предплечье + кисть; появляется из-за дальнего края при тапе, «бросает» зёрна. Нужны 2–3 фазы (появление / бросок / уход) или один слой для motion-трансформа.
5. **Зерно** — обычное (амбер) и **золотое** (сияющее) — для летящих частиц и HUD.
6. **Брызги кофе** — 4–6 мелких капель для splash при тапе.
7. **Пар** — 3 завитка (можно как маска/спрайт для зацикленного подъёма).
8. **Тени** — мягкая тень под блюдцем; контактная тень чашки.
9. **Фон-эмблема** — круг с горами/деревом/кофейной средой (приглушённый, не перегружать).

## Состояния маскота

- **idle** — чашка «дышит», пар поднимается, ноги слегка качаются, кофе колышется, мягкий блик.
- **tap** — сжатие+раскачка чашки, вскидывание ног, всплеск кофе + брызги, вылет руки, бросок зёрен.
- **combo/burst** — усиленная амплитуда, больше брызг/зёрен, бейдж «Комбо ×N».
- **tired (мало энергии)** — приглушённые цвета/анимация, «уставшая» поза, без пара.

## Анимации (motion system)

- **A. Idle** — медленно, плавно, малозаметно (2.5–5.5 c циклы).
- **B. Tap impact** — быстро, упруго (~0.5 c), spring-отклик.
- **C. Reward particles** — лёгкие зёрна по дуге к счётчику, читаемые «+N».
- **D. Splash** — аккуратные кофейные брызги, не перекрывают экран.
- **E. Limbs** — ноги: swing/jiggle; рука: quick pop-out / toss / retract.

## Форматы

- Вертикальный экран, мобильный. Маскот ~ 320px по ширине (2–3× для ретины).
- Предпочтительно SVG (масштабируемость) либо PNG@3x с прозрачностью.
- Слои именовать по назначению (`cup`, `coffee`, `leg-left`, `leg-right`, `arm`, `bean`, `bean-gold`, `splash`, `steam`, `shadow`, `emblem-bg`), чтобы заменить временные SVG-группы 1:1.
